# Wordle Image Designer

Design a 5×6 Wordle grid and get a valid word sequence that matches your pattern.

**[Open the tool](https://ppoiuy.github.io/wordle-image-designer/)**

## How it works

1. Set a target word (the answer)
2. Paint the grid with green, yellow, and white tiles
3. Click **Generate** to find real words that produce your exact pattern
4. Mark a row all-green to set it as the winning guess — rows after it are locked

## Features

- Drag to paint multiple tiles at once
- Swap colors across the entire grid
- Flip the grid horizontally or vertically
- Save and load grid presets
- Undo / redo (Ctrl+Z / Ctrl+Y)
- Session is saved automatically between visits
- Hotkeys: `1`/`Q` white, `2`/`W` yellow, `3`/`E` green
